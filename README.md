# 받아쓰기 놀이터 (1차 프로토타입)

아이들이 즐겁게 사용할 수 있는 **정적 웹 받아쓰기 게임**입니다.
핵심 UX는 크게 만든 **"문제 불러줘" 재생 버튼 인터랙션**에 맞춰 구현했습니다.

## 파일 구조

```bash
/
├─ index.html     # 화면 구조(헤더/친구 선택/문제 카드/요약)
├─ style.css      # 파스텔 톤, 카드형 레이아웃, 재생 버튼 애니메이션
├─ app.js         # 상태 관리, localStorage, 문제/점수/랭킹 로직
└─ README.md      # 실행/배포/확장 가이드
```

## 파일별 설명

- `index.html`
  - 앱 UI의 기본 뼈대를 담당합니다.
  - 영역: 상단 헤더, 친구 선택, 문제 카드(재생 버튼), 입력/제출, 하단 요약.
- `style.css`
  - 파스텔 색상, 둥근 버튼/카드, 반응형 레이아웃을 담당합니다.
  - 재생 중 상태를 강조하는 pulse/ripple 애니메이션 포함.
- `app.js`
  - 더미 문제 5개를 기반으로 받아쓰기 흐름을 처리합니다.
  - `localStorage`에 친구 목록, 선택 친구, 점수/연속정답 등 상태를 저장합니다.

## 실행 방법

1. 이 저장소를 내려받습니다.
2. 루트의 `index.html`을 브라우저에서 바로 엽니다.
3. 또는 간단한 로컬 서버를 켭니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.

## GitHub Pages 배포 방법

1. GitHub 저장소에 코드 push.
2. GitHub 저장소의 **Settings → Pages** 이동.
3. **Build and deployment**에서:
   - Source: `Deploy from a branch`
   - Branch: `main` (또는 사용 브랜치) / folder: `/ (root)` 선택
4. 저장 후 발급된 Pages URL 접속.

정적 파일(`index.html`, `style.css`, `app.js`)만으로 동작하므로 별도 빌드가 필요 없습니다.

## 2차 확장 포인트

1. **문제 세트 확장**
   - 난이도(유아/초등), 주제(동물/학교/자연)별 문제팩 분리
2. **음성 고도화**
   - Web Speech API(TTS) 또는 음성 파일(mp3) 재생으로 실제 듣기 품질 향상
3. **학습 기록 고도화**
   - 날짜별 학습량, 정답률, 취약 단어 분석
4. **게임화 강화**
   - 스티커/배지, 레벨 시스템, 주간 챌린지
5. **접근성/UX 개선**
   - 더 큰 터치 영역, 음성 안내, 오답 복습 모드
