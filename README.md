# 아이들을 위한 받아쓰기 놀이터 (5차 안정화 + 고도화)

정적 웹앱(HTML/CSS/Vanilla JS) 기반으로 GitHub Pages에서 바로 동작하는 받아쓰기 게임 서비스입니다.

## 1) 5차 최종 파일 구조

```bash
/
├─ index.html
├─ style.css
├─ data.js
├─ storage.js
├─ game.js
├─ ui.js
├─ tts.js
├─ app.js
└─ README.md
```

## 2) 현재 4차 오류 원인 요약

1. 뷰 전환 시 재생/타이머 상태 정리 부족
2. 버튼 그룹 레이아웃 규칙 부족(모바일에서 붙어 보임)
3. 비어 있는 상태 값(`-`)이 사용자 경험을 깨뜨림
4. 재생 중 뷰 이동/중복 클릭 처리 미흡
5. 세션/기록/오답 저장 정렬 및 제한 일관성 부족
6. README와 실제 구현 상태 불일치 가능성

## 3) 5차에서 수정한 버그(안정화)

- 뷰 전환 시 TTS/타이머 중단 (`switchView`에서 정리)
- 재생 중 중복 클릭 방지 및 종료 상태 정상화
- auto-next/next 흐름 정리(중복 반영 방지)
- 세션 종료 후 결과 화면 재진입 안정화
- 오답 단어 중복 누적 방식 정리(count 증가)
- 최근 세션 정렬/개수 제한(최신순 30개)
- empty state 안내 문구 전면 보강
- 버튼/카드/텍스트 간격 및 반응형 정렬 보정
- 설정 변경 즉시 반영/저장 및 confirm 기반 초기화 처리

## 4) 5차 고도화 항목

### 게임 완성도
- 일일 퀘스트 + 주간 퀘스트 진행률 저장
- 레벨 + 경험치(xp) 반영
- 레벨 칭호 표시(새싹 탐험가 ~ 받아쓰기 챔피언)
- 문제집 완주 보상 리본, 복습 완벽 성공 메시지
- 결과 화면 별점(1~3성)

### 콘텐츠/모드
- 문제집 카드에 난이도/문제수/추천대상 표시
- 복습 모드 데이터 없으면 비활성 + 안내
- 문제 타입 확장(`word`, `sentence`) 및 긴 문장 UI 대응
- 자유/챌린지/스피드 모드 분기 로직 유지/강화

### 보호자 보기
- 리포트에 코멘트(약한 단어 기반) 추가
- 약한 단어 TOP, 최근 5세션 요약 가독성 개선

### 설정
- 자동 다음 문제 / 음성 속도 / 효과 on/off / 기본 플레이어
- 샘플 초기화 / 전체 초기화 버튼 + confirm

## 5) TTS fallback 방식

`tts.js` 동작 순서:
1. `speechSynthesis` 지원 여부 확인
2. 한국어(`ko`) 음성 우선 선택 후 재생
3. 미지원/오류 시 fallback 타이머로 재생 완료 이벤트 처리
4. 뷰 이동/중단 시 `AppTTS.stop()`으로 즉시 취소

## 6) localStorage 버전/마이그레이션

- 현재 저장 키: `dictationPlayground_v5`
- 이전 버전 키: `v4`, `v3`, `v2`
- 로딩 우선순위:
  1. v5
  2. v4 → v5 변환
  3. v3 → v5 변환
  4. v2 → v5 변환
  5. 없으면 샘플 데이터 생성
- 누락 필드는 `normalize`에서 자동 보정
- 주요 필드: `selectedBookId`, `selectedMode`, `settings.voiceRate`, `weakWords`, `quests.daily`, `quests.weekly`, `recentSessions`, `achievements`

## 7) GitHub Pages 배포 체크리스트

1. 홈/문제집/플레이/결과/랭킹/업적/리포트/설정 이동 확인
2. TTS 지원 브라우저에서 실제 음성 재생 확인
3. TTS 미지원 환경에서 fallback 동작 확인
4. 스피드 모드 시간 종료 시 자동 오답 처리 확인
5. 복습 모드 빈 상태 비활성 카드/안내 문구 확인
6. 세션 완료 후 결과/재시작/문제집 이동 흐름 확인
7. 리포트 데이터(약한 단어/최근 세션) 표시 확인
8. 설정 변경 후 새로고침 유지 확인
9. v4/v3/v2 데이터 보존 마이그레이션 확인

## 8) 수동 테스트 시나리오

1. 새 친구 추가
2. 문제집 선택(예: 학교 생활)
3. 챌린지 모드 시작
4. `문제 불러줘`로 TTS 재생
5. 정답/오답 제출 및 피드백 확인
6. 한 번 더! 재생 횟수 제한 확인
7. 세션 완료 후 결과 별점/보상 확인
8. 리포트 뷰에서 아이별 지표 확인
9. 설정에서 음성 속도 변경 후 재생 반영 확인
10. 샘플/전체 초기화 버튼 확인

## 9) 실행 방법

```bash
python3 -m http.server 8000
```
브라우저에서 `http://localhost:8000`

또는 `index.html` 직접 열기.
